# Mini Games Collection

🎮 多合一小游戏平台，支持模块化构建和现代化开发流程。

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
# 启动开发服务器（支持热重载）
npm run dev

# 或者直接打开 index.html
```

### 构建生产版本
```bash
# 构建所有游戏
npm run build

# 构建单个游戏
npm run build:math-rain
npm run build:tetris
npm run build:tank-battle


# 预览构建结果
npm run serve
```
